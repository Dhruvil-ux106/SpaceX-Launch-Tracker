# 🚀 SpaceX Launch Tracker

## 📌 Project Description

The SpaceX Launch Tracker is a web application that allows users to explore SpaceX rocket launches using real-time data from a public API. Users can view details about past and upcoming launches, including mission names, launch dates, and success status.

## 🎯 Purpose

The purpose of this project is to demonstrate the use of JavaScript, API integration, and modern UI development by building an interactive and responsive web application.

## 🔗 API Used

SpaceX API
https://api.spacexdata.com/v4/launches

## ✨ Planned Features

### 🔍 Search

* Search launches by mission name

### 🔽 Filter

* Filter launches based on:

  * Upcoming launches
  * Past launches
  * Successful launches

### 🔼 Sort

* Sort launches by:

  * Date (ascending/descending)
  * Mission name (A-Z / Z-A)

### ⭐ Interactive Features

* Mark launches as favorites
* View detailed information about each mission

### 🎨 UI Features

* Dark space-themed UI
* Responsive design (mobile, tablet, desktop)
* Card-based layout for launches

## 🛠 Technologies Used

* HTML
* CSS (or Tailwind/Bootstrap)
* JavaScript (ES6)
* Fetch API

## 🚀 How to Run the Project

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
